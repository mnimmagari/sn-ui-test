What are Learning Plans?

A Learning Plan is a route taken by a learner through a set of modules to build knowledge and skills.

How many Learning Plans can I work at one time?

You can start and work on as many Learning Plans as you'd like. There is overlap between Learning Plan content. A module you see in one Learning Plan may also be part of other Learning Plans.

How do I get Jakarta instance?

There are three ways to obtain a Jakarta instance:
  1. Select Jakarta as the version when requesting an instance.
  2. Upgrade your existing instance by opening Manage > Instance and selecting "Upgrade to Jakarta" from the Action menu. Your previous work is preserved.
  3. If you already have an instance but want to start over without preserving your work, release your instance, wait 15 minutes, then request a new instance.

Why do you need to reclaim instances

We have more registered developers than we have instances. Additionally, since the program is free, we would like these instances to be leveraged by many individuals that will use them frequently. By reclaiming instances, we are able extend a sizable investment in the developer program across a much larger group of developers.

Can I go back to the previous release once I upgrade?

There is no self-service option to downgrade release versions. You can, however, release your instance and request a new instance on the version of your choice. You may have to wait up to 15 minutes after you release your instance to request a new instance.

How do I get a Jakarta instance if it says "not available" on the release selection screen?

You can request an instance of a different release and upgrade to Jakarta. See the "How do I get a Jakarta instance" question in this section for upgrade instructions.

How can I track my Learning Plan progress?

For authenticated (logged in) users, we track your progress for you! Open the Learning Plan to see the progress indicator. Completed modules have a Resume button. Completed lessons have a green check mark in a green circle next to the lesson name. You can also find information about your Learning Plan progress in your developer site dashboard.

I started a Learning Plan and noticed that parts of it were already completed. How did that happen?

The Learning Plans are made up of modules from our training course offerings. If you work through a module as part of a course, you get credit for completing that module. If you start a Learning Plan which uses a module you already completed, you still get credit for the work you have done. You can always work the module a second time if you would like to refresh your memory.

Does the order I work on the modules in a Learning Plan matter?

The modules in a Learning Plan are presented in a suggested order. You can work the modules in the suggested order or in any order you'd prefer; you are not constrained to the suggested order.

Do I have to complete a Learning Plan after I start it?

Starting a Learning Plan adds it to your developer site user dashboard. Although you do not have to complete the Learning Plan, it will remain on your user dashboard in the in progress state.

How do I reset my Learning Plan so I can start over?

We do not want to take away credit for work you have already done. You can work though the Learning Plans as many times as you would like even if all or part show of the Learning Plan content shows as completed.
